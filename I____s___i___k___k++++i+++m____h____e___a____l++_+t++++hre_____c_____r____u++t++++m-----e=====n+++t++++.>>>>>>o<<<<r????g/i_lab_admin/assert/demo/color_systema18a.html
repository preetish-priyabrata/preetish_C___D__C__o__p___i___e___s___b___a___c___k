Color system usage
Class Prefixes Description
Background
.bg-* *-50, *-100, *-200, *-300, *-400, *-600, *-700, *-800, *-900,, *-1000, *-1100 *-a100, *-a200, *-a300, *-a400, *-a500, *-a600, *-a700, *-a800 Background color. Can be added to dropdown menus, sidebar, navbar, alerts, inputs, panel panel-colors, selects etc.
Borders
.border-* *-50, *-100, *-200, *-300, *-400, *-600, *-700, *-800, *-900,, *-1000, *-1100 *-a100, *-a200, *-a300, *-a400, *-a500, *-a600, *-a700, *-a800 4 sides border color. Any element that contains border
.border-top-* Top border only. Any element that contains all sides border or top border only
.border-bottom-* Bottom border only. Any element that contains all sides border or bottom border only
.border-left-* Left border only. Any element that contains all sides border or left border only
.border-right-* Right border only. Any element that contains all sides border or right border only
Text
.text-* *-50, *-100, *-200, *-300, *-400, *-600, *-700, *-800, *-900,, *-1000, *-1100 *-a100, *-a200, *-a300, *-a400, *-a500, *-a600, *-a700, *-a800 Text color. Can be used in text, icons and links
